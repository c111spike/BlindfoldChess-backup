Add a new table for Training Stats to keep your Game History clean.

SQL

CREATE TABLE training_stats (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    mode TEXT,          -- 'color_blitz' or 'coordinate_sniper'

);
Personal Best Logic: To display the PB on the dashboard, simply query: SELECT MAX(score) FROM training_stats WHERE mode = 'color_blitz' SELECT MIN(score) FROM training_stats WHERE mode = 'coordinate_sniper'
