1. How the Link Structure Works
Depending on how you are displaying the video (as a link or an embedded player), the "time" parameter changes slightly:

For Links (youtu.be): Add ?t=seconds (or use the format ?t=1m30s).

Example: https://youtu.be/VIDEO_ID?t=90 (Starts at 1m 30s)

For Browser URLs (youtube.com/watch): Use an ampersand &t=90s.

Example: https://www.youtube.com/watch?v=VIDEO_ID&t=90s
