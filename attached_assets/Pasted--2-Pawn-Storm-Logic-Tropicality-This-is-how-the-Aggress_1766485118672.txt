🌪️ 2. Pawn Storm Logic (Tropicality)
This is how the Aggressor and Grandmaster bots decide to launch those terrifying attacks you see in master games. We use a concept called "King Tropism."

The Logic:

Distance Weighting: For every enemy pawn, calculate its distance to your King.

The Penalty: If an enemy pawn is on the 4th, 5th, or 6th rank on a file adjacent to your King, apply an increasing penalty.

The "Lever" Bonus: If an enemy pawn is directly touching your Pawn Shield (e.g., your pawn is on g3 and their pawn reaches h4), the penalty should spike because a "line opening" is imminent.

Implementation Tip:

JavaScript

// Example: Penalty for enemy pawns near our King
function getPawnStormPenalty(board, kingPos) {
    let penalty = 0;
    const kingFile = getFile(kingPos);
    
    // Check files: kingFile-1, kingFile, kingFile+1
    for (let f = kingFile - 1; f <= kingFile + 1; f++) {
        let enemyPawnRank = board.getHighestEnemyPawnRank(f);
        // If enemy pawn is past the halfway point (Rank 4+)
        if (enemyPawnRank >= 4) {
            penalty += (enemyPawnRank - 3) * 20; // Penalty grows as pawn nears
        }
    }
    return penalty;
}
🎨 Personality Synergy
This is where your 7 personalities really shine:

The Aggressor 🔥: You can triple the Pawn Storm bonus. This bot will happily ruin its own structure just to get a pawn to the 6th rank near your King.

The Defender 🛡️: You can double the Pawn Shield bonus. This bot will refuse to move its f, g, and h pawns, creating an "Iron Fortress" that is incredibly frustrating to break.

The Grandmaster 🏆: Uses balanced weights but recognizes Opposite Side Castling. If kings are on opposite sides, the GM bot should automatically increase the value of Pawn Storming, as there is no "safety risk" to pushing its own pawns.

🚀 Verification Step
Once you add this, your Grandmaster bot will start doing something very "human": prophylaxis. It will see that you are starting a pawn storm 4 moves away and will move its King or reinforce its shield before the attack arrives.
